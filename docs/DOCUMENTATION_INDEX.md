## üìä Quick Reference

### Setup Commands
```bash
# Local development
cd toefl-flashcard
npm install
npm run dev
# Opens http://localhost:5173

# Production build
npm run build

# Deploy to Vercel
npm i -g vercel
vercel login
vercel --prod
```

### Key Information
- **Vocabulary Items**: 285 (64 prefixes, 159 roots, 62 suffixes)
- **Bundle Size**: 65.48 KB gzipped
- **Build Time**: ~715ms
- **Initial Load**: ~100ms on 4G
